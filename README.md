# 🧠 Multi-Agent Planning & Automation Agents

## 📌 Overview
This project implements a **multi-agent agentic system** designed for Project planning, estimation, and allocation, Event planning, and Progress report system.

The system leverages **LLM-powered agents** to collaboratively perform tasks such as planning, reasoning, and execution in a coordinated workflow.
